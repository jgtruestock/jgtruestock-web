📋 一、網站是什麼

JGTrueStock 是一個給 JG 會員使用的投資追蹤平台，有兩個部分：

  • 前台（會員看）：JG 提過的股票記錄 + 每支股票的法說會比對分析 + 近期新聞
  • 後台（JG 管）：新增/刪除提股、管理法說會點評、追蹤大神頻道內容

網站網址：jgtruestock-web.vercel.app

點擊任一支股票後進入的詳情頁，有兩個區塊：

法說會 AI 分析邏輯（內部技術說明）

AI 生成點評分三個步驟：

  1. 抓資料：從 FMP API 取最新法說會逐字稿 + 最新 30 篇新聞
  2. 生成主體：Claude Sonnet 根據三段式格式生成法說會方向 / 官方動態 / 比對結論
  3. 結構化要點：同時提取 7 類要點（revenue、margin、capex、product、headcount、guidance、market_expansion），每個要點附上 AI 判斷的狀態（fulfilled / partially / broken / pending / unclear）
✅ 63 支股票目前全部已自動生成草稿並發布（2026-07-07 批次完成）

重要技術決定

  • Vercel 部署不是自動觸發（需要透過 Vercel API 手動觸發），因為 GitHub 帳號連結問題
  • AI 法說會生成改成非同步（Async Job Queue）：後台按按鈕 → 馬上回應 → Mac 本地 worker 執行 → 完成自動刷新
  • 股票詳情頁有 ISR 快取（1小時），支援幾千名同時用戶
  • API 回應快取 5 分鐘（Vercel CDN Edge Cache）
